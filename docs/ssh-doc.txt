1.   УСТАНОВКА SSH-КЛЮЧА.
1.1. Проверить наличии SSH-ключей:  ls -al ~/.ssh 
1.2. Запустить генерацию нового ключа: ssh-keygen -t rsa -b 4096 -C "akataevakultaj@gmail.com"
1.3. Проверить наличие SSH-ключей:  ls -al ~/.ssh

2.   Добавление SSH-ключа в ssh-agent.
2.1. Запустить ssh-agent в фоновом режиме:   eval "$(ssh-agent -s)"
2.2. Добавить новый SSH-ключ в ssh-agent:  ssh-add ~/.ssh/id_rsa

3.   Копирование SSH-ключа ( для установки в настройках GitHub ). 
3.1. Вариант 1. Вывести в терминал SSH-ключ:  cat ~/.ssh/id_rsa.pub
     Скопировать в буфер обмена для вставки в настройки GitHub.
3.1. Вариант 1. Вывести в терминал SSH-ключ:  notepad ~/.ssh/id_rsa.pub 
     Скопировать в буфер обмена для вставки в настройки GitHub.

4.   Установка SSH-ключа в настройках GitHub.
4.1. Перейти в: аккаунт GitHub → Avatar → Settings → SSH and GPG keys → New SSH key 
     Вставить SSH-ключ в поле Key и дать ему название, например - "My local computer". 
     Кликнуть Add SSH key.
4.2. Подтвердить доступ паролем.
     В окне ввести пароль, кликнуть Confirm.
4.3. Наблюдать результат.
